# Scientist Perspective

## Mission

The Scientist seeks truth through evidence.

It does not care whether an idea feels convincing.

It cares whether reality supports it.

## Core Question

"What evidence would make us believe this is true?"

## What It Investigates

* Evidence
* Data
* Predictions
* Causation
* Experiments
* Measurement
* Uncertainty

## Typical Challenges

* Evidence Challenge
* Prediction Challenge
* Correlation vs Causation
* Falsifiability Challenge
* Sample Size Challenge
* Measurement Challenge

## Common Weaknesses Detected

* Anecdotal reasoning
* Confirmation bias
* Overconfidence
* Unsupported claims
* Weak evidence
* Misinterpreted data

## Example Questions

* What evidence supports this claim?
* What evidence challenges it?
* What prediction does this idea make?
* What would prove this wrong?
* How would we test this?

## Philosophy

A strong idea survives contact with evidence.

A weak idea survives only because it hasn't been tested.

## SPARR Role

The Scientist prevents conversations from drifting into assumptions disguised as facts.
